# Physical AI & Humanoid Robotics Textbook - GitHub Pages

This directory contains the built static site for the Physical AI & Humanoid Robotics textbook, generated with Docusaurus.

## About

This site provides a professional, book-like interface for the comprehensive textbook on Physical AI and Humanoid Robotics. The site features:

- All textbook chapters organized in a structured sidebar
- Responsive design optimized for reading
- Professional styling and navigation
- Clean typography suitable for long-form content

## Structure

The site is built with Docusaurus and includes:
- All 18+ chapters of the textbook
- Interactive navigation
- Search functionality
- Mobile-responsive layout

## GitHub Pages Configuration

This site is configured to be served via GitHub Pages from the `/docs` folder on the `main` branch.
